# Windows 日志与事件查看器

一个功能全面的 Windows 系统诊断工具，帮助普通用户快速了解电脑运行状态、安全状况和异常活动，无需学习复杂的事件查看器。

## 功能

### 🔍 异常检测（一键诊断）
自动分析六个维度的系统数据，给出 **0-100 分风险评分**：

- **登录失败记录** — 检测是否有人尝试破解密码
- **深夜登录 (22:00-06:00)** — 谁在非工作时间用过电脑
- **远程桌面登录** — 是否有人远程连接过
- **USB 设备记录** — 历史上插过哪些 U 盘
- **系统崩溃/异常关机** — 有没有蓝屏、突然断电
- **活跃网络连接** — 当前哪些进程连接到外部地址
- **最近文件变动** — 指定日期修改过哪些文件
- **安全建议** — 根据风险等级给出具体操作建议

### 📊 日志查看（10 个标签页）

| 标签页 | 内容 |
|---|---|
| 异常检测 | 综合安全分析报告 + 风险评分 |
| 系统日志 | 系统事件（崩溃、服务故障、关机原因） |
| 安全日志 | 登录/注销、权限变更、账户操作 |
| 应用日志 | 程序错误、安装/卸载记录 |
| 登录记录 | 谁登录过、什么方式、什么时间 |
| USB 历史 | 插过哪些 USB 设备 |
| 文件改动 | 指定日期修改了哪些文件 |
| 进程历史 | 运行过什么程序、谁运行的 |
| 网络连接 | 当前活跃的 TCP 连接 |
| 已装软件 | 安装的软件列表 |
| 系统信息 | 主机名、CPU、内存、磁盘、服务等 |

### 🎛️ 操作功能

- **日期选择** — 支持最近 30 天任意一天，精确到日
- **搜索过滤** — `Ctrl+F` 实时过滤所有表格数据
- **列排序** — 点击表头按任意列排序
- **双击详情** — 双击任意行查看完整信息
- **导出 CSV/TXT** — 右键导出当前页数据
- **刷新** — `Ctrl+R` 一键刷新当前标签页
- **事件含义翻译** — 事件 ID 自动翻译成中文含义

## 系统要求

- Windows 7 SP1 及以上（x64）
- 无需安装 Python
- 无需安装任何依赖
- **建议：以管理员身份运行**（可获取完整安全日志）

## 使用方法

### 运行

1. 下载 `Windows日志事件查看器.exe`
2. **右键 → 以管理员身份运行**（推荐，否则部分安全日志无法读取）
3. 点击左侧标签页查看各类数据
4. 使用顶部日期下拉框选择要查看的日期

### 快速检查电脑是否安全

打开程序 → 查看第一个标签页「🔍 异常检测」→ 看风险评分：

- **🔴 高风险 (≥60 分)** — 立即改密码，检查是否有不明软件
- **🟡 中等风险 (30-59 分)** — 存在值得关注的异常
- **🟢 低风险 (10-29 分)** — 少量异常，建议留意
- **✅ 正常** — 未发现异常

## 从源码运行

```bash
# 安装 Python 3.9+
# 进入项目目录
cd 日志查看器

# 直接运行（无需安装第三方库）
python log_viewer.py
```

## 自行打包

```bash
pip install pyinstaller
pyinstaller --noconsole --onefile --name "Windows日志事件查看器" log_viewer.py
```

## 技术说明

- **语言**: Python 3.9+
- **GUI 框架**: tkinter（Python 内置）
- **数据来源**: Windows 事件日志 (Security/System/Application)、注册表、WMI
- **数据获取**: 通过 PowerShell 调用 Get-WinEvent 等 cmdlet
- **打包工具**: PyInstaller
